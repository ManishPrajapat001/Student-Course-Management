1. What is JDK, JRE, JVM
   JDK: Java development kit,It contains JRE & JVM and other development tools.It as whole is used for java development.It contains java compiler which convert code to bytecode    
   JRE: Java run environment,it includes JVM and core java libraries.It is used to run java application.    
   JVM : Java virtual machine,It run byte code in the machine and gives result.


2. What is bytecode     
It is code generated by javacompiler,It is platform independent code which is run on jvm .This is stored in .class file.Because bytecode is the same everywhere, Java programs can run on any system with a JVM — this is what gives Java its “Write Once, Run Anywhere” nature.



3. What does “write once, run anywhere” mean 
    bytecode generated by java compiler is same on every machine.That's why bytecode is platform independent code. This byte code(Java programs) can run on any system with a JVM .
